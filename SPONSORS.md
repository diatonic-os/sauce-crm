# Sponsor Sauce CRM

Sauce CRM is open source (MIT) and built in the open. There is no paywalled
tier, no "Plus" license, and no telemetry — the whole plugin is free and stays
free. If it has saved you time wrangling your relationships, sponsoring keeps
the roadmap moving and the integrations maintained.

Sponsorship is entirely optional and never gates a feature. Per our community
guardrails, there are **no donation prompts inside the plugin UI** — this page
and the README are the only places we ask.

## How to sponsor

- **GitHub Sponsors:** [github.com/sponsors/iamdrewfortini](https://github.com/sponsors/iamdrewfortini)
- **Buy Me a Coffee** and **Patreon** links are surfaced from the repo's
  Sponsor button (see [`.github/FUNDING.yml`](.github/FUNDING.yml)).

## Tiers

Each tier is cumulative — every tier includes the perks of the ones below it.

### ☕ Supporter — $5 / month

> Channels: GitHub Sponsors, Buy Me a Coffee

- Sponsor badge next to your name in the README Sponsors section
- `#supporters` role in the community Discord

### 🌶️ Sponsor — $25 / month

> Channels: GitHub Sponsors, Patreon

- Everything in **Supporter**, plus:
- Access to the BRAT beta channel — install pre-release builds before they ship
  to the community store
- Early roadmap input via a quarterly sponsor survey

### 🔧 Contributor — $100 / month

> Channels: GitHub Sponsors, Patreon

- Everything in **Sponsor**, plus:
- Your name listed in this `SPONSORS.md`
- A quarterly 30-minute roadmap call

### 🛠️ Maintainer — $500 / month

> Channels: GitHub Sponsors

- Everything in **Contributor**, plus:
- Your logo in the README
- Priority placement in the integration-request queue
- A direct support channel
